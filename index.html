Hotels API

Cet api petmet de retourner tout les hotels de la base de donnée en format JSON

la table Hotels contient

  • id
  • nom
  • description
  • adresse
  • telephone
  • email
  • reseaux_sociaux


Images API

Cet api petmet de retourner toutes les images des hotels de la base de donnée en format JSON

la table Images contient

  • id_img
  • id-hotel: clé etrangere, c'est l'id d'un hotel
  • url: lien vers l'image, chaque hotel contient 4 images
